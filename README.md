# SimbaView  
Visualizador Local de Arquivos de Afastamento de Sigilo Bancário (SIMBA).

Este projeto consiste em uma ferramenta **100% local**, em um único arquivo HTML, que permite visualizar e correlacionar os 5 arquivos previstos na [Carta-Circular BACEN nº 3.454/2010](https://www.bcb.gov.br/estabilidadefinanceira/exibenormativo?tipo=Carta%20Circular&numero=3454):

- **AGÊNCIAS**
- **CONTAS**
- **TITULARES**
- **EXTRATO**
- **ORIGEM_DESTINO**

A ferramenta não envia dados para nenhum servidor — tudo é processado apenas no navegador.

---

## Como usar

1. [Baixe o arquivo](https://github.com/jesobreira/simbaview/blob/main/index.html)

2. Abra o arquivo no seu navegador (Chrome, Edge, Firefox ou Safari).  
   Nenhuma instalação é necessária.

3. Carregue os 5 arquivos TXT (cada um no campo correspondente).

4. Clique em **"Carregar"** para importar todos de uma vez.

5. Use o menu de visualização para alternar entre:
   - Agências  
   - Contas  
   - Titulares  
   - Extrato  
   - Origem/Destino
   
---

## Privacidade

A página funciona **inteiramente offline**, sem utilizar APIs externas.  
Para segurança máxima, recomenda-se salvar o arquivo HTML e acessá-lo sem conexão com a internet.
